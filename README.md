# CRUD SAP CAP

Projeto simples de CRUD desenvolvido utilizando **SAP CAP (Cloud Application Programming Model)** com interface gerada pelo **Fiori Elements nativo do CDS**.

## 📌 Objetivo

Este projeto foi criado com o objetivo de praticar:

- Desenvolvimento backend com SAP CAP
- Modelagem de entidades utilizando CDS
- Exposição de serviços OData
- Integração com interface Fiori Elements
- Estruturação de projeto Node.js dentro do ecossistema SAP

## 🛠 Tecnologias utilizadas

- SAP CAP (Node.js)
- CDS (Core Data Services)
- SAP Fiori Elements
- SQLite (ambiente local)

## ⚙️ Funcionalidades

- Criação de registros
- Leitura de registros
- Atualização de registros
- Exclusão de registros

## 🚀 Como executar o projeto

npm install


2. Executar a aplicação:

cds watch


3. Acessar no navegador:

http://localhost:4004
